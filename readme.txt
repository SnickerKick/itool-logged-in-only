A Plugin to lock down the whole site to prevent public access.

== Description ==

Only logged-in users can view the site if this plugin is activated.

== Installation ==

1. Upload `logged-in-only.php` to the `wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Your site is locked down to the public.
